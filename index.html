เขียนเว็บไซต์ด้วยภาษาHTML

รู้จักกับHTML

เนื่องจาก HTML ได้เริ่มใช้กันมานานตั้งแต่ต้นทศวรรษที่ 19 และพัฒนาต่อเนื่องจนมาหยุด ที่เวอร์ชัน 4.01 แต่ด้วยเทคโนโลยีและแอพพลิเคชันที่เกี่ยวข้องกับอินเทอร์เน็ตมีการพัฒนาแบบ ก้าวกระโดด
มีการใช้งานสื่อสังคมออนไลน์อย่างกว้างขวาง เราสามารถทวีตข้อความโต้ตอบกันผ่าน ทาง Twitter, สื่อสารด้วย Facebook, ฝากรูปกับ Instragram ฯลฯ HTML 4.01 ไม่ได้ถูกออกแบบ มารองรับตั้งแต่แรก จึงมีการพัฒนาส่วนเสริม (Extension หรือ Add-on หรือ Plug-in) มาติดตั้ง เพิ่มเข้าไป เพื่อให้รองรับกับเทคโนโลยีใหม่ๆ ได้
จนมาถึงเดือนตุลาคม 2014 องค์กรกลางที่ดูแลด้านมาตรฐานอินเทอร์เน็ต (WBC) ก็ได้ ประกาศใช้งาน HTML5 อย่างเป็นทางการ การรอคอยกว่า 14 ปี ก็สิ้นสุดลง
HTML5 เป็น HTML รุ่นใหม่ที่มีคุณสมบัติครบถ้วน ในแบบที่ไม่ต้องพึ่ง Extension มากนัก สามารถรองรับเบราว์เซอร์ได้ทุกแบบ สามารถทำงานบ้านระบบ (Platform) ได้อย่างไม่มีปัญหา สามารถใช้กับอุปกรณ์อื่นๆได้ละพวกขึ้นโดย
เฉพาะกับโทรศัพท์มือถือ หรืออุปกรณ์ที่มีหน้าจอหลาย ขนาด เช่น แทบเล็ต (Tablet)

ความเป็นมาของเว็บ

เว็บ (Web) เริ่มปรากฏโฉมให้เราได้ใช้งานกันในตอนต้นของทศวรรษ 90 โดย Web1.0 ได้ ถือกำเนิดขึ้นเป็นเวอร์ชันแรก โดยในเวอร์ชันนี้ เว็บเพจจะเป็นแบบธรรมดาที่สุด ไม่มีการโต้ตอบ มีลักษณะเป็นเว็บแบบ Static(คงที่)
ที่ใช้แสดงข้อมูลเป็นหลัก เช่น แสดงข้อความ รูปภาพ เป็นต้น โดยทั่วไปข้อมูลที่ปรากฏบนเว็บ จะไม่ถูกเปลี่ยนแปลงบ่อยๆ เราอาจจะพบข้อความเต็มบนเว็บเป็น สัปดาห์ หรือบางทีเป็นเดือนก็มี
ข้อมูลที่แสดงบน Web1.0 จะถูกเขียนขึ้นโดยนักพัฒนาหรือผู้ที่ทำหน้าที่ดูแลเว็บโดยตรง ผู้ใช้ไม่สามารถเข้าไปแก้ไขหรือปรับปรุงข้อมูลบนเว็บได้
ในเดือนตุลาคมปี 1992 มีเว็บไซต์จำนวน 26 เว็บไซต์ บุกเบิกด้วยการนำข้อมูลไปแสดง บน WWW เป็นครั้งแรก และอีก 3 ปีถัดมา คือในปี 1995 สามารถฟังเสียงในลักษณะ Streaming
ซึ่งถือว่าเป็นการฟังเสียงในแบบออนไลน์ได้เป็นครั้งแรก
ปีถัดมาคือ ปี 1996 มีผู้คนเข้าใช้งานอินเทอร์เน็ตกว้างขวางขึ้นกว่า 16 ล้านคน และในปี 1999 จำนวนคนที่ใช้งานอินเทอร์เน็ตก็เติบโตมากถึง 147 ล้านคนทั่วโลก หลังจากนั้นในปี 1999 ก็มีผู้เสนอการใช้บล็อก (blog) ในการสื่อสาร
และในปี 2004 ก็ทำให้บล็อกกลายเป็นการสื่อสารหลัก บนอินเทอร์เน็ต
ปี 2005 มีการเสนอ AJAX ซึ่งเป็นเทคนิคในการอัพเดตข้อมูลบางส่วนของเว็บ โดยไม่จำเป็น ต้องโหลดเว็บเติมใหม่ทั้งหมด ทำให้ภาษาต่างๆ ไม่ว่าจะเป็น PHP, ASP, ASP.Net และภาษาอื่นๆ ได้รับ
อานิสงส์ของการเขียนเว็บแนวใหม่แบบ AJAX ไปด้วย
Web 2.0 ก็ได้ปรากฏตัวขึ้น ปี 2006 โดยในปีนี้มีจำนวนเว็บไซต์เพิ่มมากถึง 92 ล้านเว็บไซต์ และมีผู้ใช้อินเตอร์เน็ตเกีอบพันล้านคน ปีถัดมาคือ ปี 1996 มีผู้คนเข้าใช้งานอินเทอร์เน็ตกว้างขวางขึ้นกว่า16ล้านคน
และในปี 1999 จำนวนคนที่ใช้งานอินเทอร์เน็ตก็เติบโตมากถึง 147 ล้านคนทั่วโลก หลังจากนั้นในปี 1999 ก็มีผู้เสนอการใช้บล็อก (blog) ในการสื่อสาร และในปี 2004 ก็ทำให้บล็อกกลายเป็นการสื่อสารหลัก บนอินเทอร์เน็ต

Text editor สำหรับแก้ไขโค้ด

เครื่องมือในการสร้างหรือแก้ไขเว็บเพจ คือ Text editor ซึ่งใช้สำหรับแก้ไขโค้ด ไม่แนะนำให้ ใช้ Notepad หรือ Microsoft Word แต่ควรจะเป็นโปรแกรมที่ถูกออกแบบมาสำหรับการป้อนโค้ด HTML โดยเฉพาะ เช่น Notepad++,vscode
EditPlus, Aptana, Dreamweaver, E-Text editor, Intype หรือจะใช้ UltraEdit ก็ได้ Text editor บางตัวจะมีระบบช่วยเหลือให้เราเขียนโค้ด HTML ได้ง่ายขึ้น เช่น หากเรา ใส่แท็กเปิดอย่าง <body> โปรแกรมก็จะช่วยใส่แท็กปิด </body> มาให้โดยอัตโนมัติ มีการใช้ สีที่แตกต่างกันระหว่างแท็ก ตัวแปร ข้อความ ทำให้สามารถอ่านโค้ดได้ง่าย นอกจากนั้นยังสามารถ แสดงหรือซ่อนโค้ดบางส่วนไว้ชั่วคราว สามารถใส่คำอธิบาย (Comment) ด้วยคีย์ลัด และที่สำคัญ ยังมีระบบอัตโนมัติที่ช่วยเหลือในระหว่างการพิมพ์โค้ด เช่น แสดงชื่อตัวแปรที่เคยใช้งานมาก่อน หน้านี้ แนะนำชื่อฟังก์ชัน หรือแอดทริบิวต์ต่างๆ
