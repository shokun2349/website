เอลิเมนต์

          เอลิเมนต์ (element) คือส่วนประกอบที่เป็นหน่วยย่อยของโครงสร้าง HTML ในแต่ละ เอลิเมนต์จะใช้แท็ก (tag) เพื่อบอกให้ทราบว่าแต่ละเอลิเมนต์มีหน้า และมีโครงสร้างเป็นเช่นไร

การใช้แท็ก

          แท็กแรกที่จะกล่าวถึงคือ แท็ก ❬html❭ เป็นแท็กที่ใช้บอกขอบเขตของโค้ด HTML เป็นการ ประกาศให้ทราบว่าข้อความทั้งหมดที่อยู่ภายในแท็กนี้คือโค้ด html ซึ่งเบราว์เซอร์จะทำหน้าที่แปล ความหมายของโค้ดนี้เพื่อแสดงผลเป็นเว็บเพจต่อไป เอลิเมนต์ html จะประกอบด้วย แท็กเปิด (start tag) เพื่อบอกจุดเริ่มต้น และแท็กปิด (end tag) เพื่อบอกจุดสิ้นสุดของเอลิเมนต์ ดังนี้
