เขียนเว็บไซต์ด้วยภาษาHTML

เตรียมพร้อมก่อนใช้งานHTML5

HTML5 ได้ถูกปรับปรุงให้รองรับกับแอพพลิเคชัน ซึ่งเป็นที่นิยมในปัจจุบัน เช่น สามารถเล่น ไฟล์เสียง หรือวิดีโอได้ โดยที่ไม่จำเป็นต้องติดตั้ง Extension
หรือวิดีโอได้ โดยที่ไม่จำเป็นต้องติดตั้ง Extension ใช้ โดยที่ไม่ต้องเสียเวลาดาวน์โหลดสิ่งเดิมๆ ซ้ำตลอด เวลา ฯลฯ อาจกล่าวได้ว่า HTML5 ถูกสร้างขึ้นมาเพื่อการเป็นมาตรฐานกลาง เพื่อรองรับการใช้งาน เว็บเพจในยุคปัจจุบันนั่นเอง

สิ่งใหม่ๆที่พบใน HTML5

          สิ่งแรกที่ทุกคนตั้งคำถามเกี่ยวกับ HTML5 คือ อะไรคือ HTML5 และคำถามที่ตามมาคือ แล้วมีอะไรใหม่ๆ หรือมีอะไรดีบ้างใน HTML5 และตีความหมายโค้ดในแบบ HTML5

ประการแรก

          ให้สังเกตที่บรรทัดแรกของ HTML จะต้องมีคำว่า ❬!DOCTYPE HTML❭ ที่ใช้บนเว็บเพจนั้นมีรูปแบบที่สั้นลง โดยกำหนด เป็นการบอกให้เบราว์เซอร์ทางานแบบ Standard mode

ประการที่สอง

          รูปแบบของ Character set เพียง ❬ charset=UTF-8" ❭ลงในส่วนของแท็ก ❬head❭❬/head❭

ประการที่สาม

          HTML5 มีการจัดโครงสร้างของเว็บเพจใหม่ โดยจะมีแท็กที่ใช้บ่งบอกถึง โครงสร้าง เช่น มีแท็ก ❬nav:❭ แสดงลิงก์หลักของเว็ปไซต์ มีแท็ก ❬header❭ สำหรับแสดงข้อมูลในส่วนหัว มีแท็ก ❬aside❭ แสดงข้อมูลในส่วนด้านข้าง มีแท็ก ❬footer❭, แสดงข้อมูลในส่วนต่างของ เว็บเพจ มีแท็ก -section สำหรับแบ่งข้อมูลออกเป็นส่วนๆ หรือแท็ก carticle ที่แยกรายละเอียด รองข้อมูลอนใน เป็นต้น
